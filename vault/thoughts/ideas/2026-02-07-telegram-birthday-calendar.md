## Context
Голосовая заметка: у контактов в Telegram есть дни рождения, хочется их отобразить в календаре.

## Insight
Автоматизация синхронизации дат рождения из мессенджера в календарь экономит время и помогает не забывать о важных датах.

## Implication
Можно автоматизировать через Telegram API (MTProto/Bot API) + Google Calendar API. Возможно, уже существуют готовые боты или скрипты.

## Next Action
Исследовать доступные инструменты и API для экспорта birthday-данных из Telegram-контактов в календарь.
